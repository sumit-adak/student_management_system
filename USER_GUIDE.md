## Getting Started

### First Time Access

**Important**: The first user to register will automatically become an administrator!

1. Open the application in your browser
2. You'll be redirected to the login page
3. Click "Don't have an account? Sign up"
4. Enter a username (letters, numbers, and underscores only)
5. Create a password (minimum 6 characters)
6. Confirm your password
7. Click "Sign Up"
8. You'll be automatically logged in

### Logging In

1. Enter your username
2. Enter your password
3. Click "Sign In"
4. You'll be taken to the Dashboard

### Logging Out

- Click the **"Logout"** button at the bottom of the sidebar (desktop)
- Or in the mobile menu (mobile devices)

## User Roles

### Regular User
- Access to all student management features
- Can add, view, and manage students
- Can record attendance
- Can manage fees

### Administrator
- All regular user permissions
- Access to Admin Dashboard
- Can manage user roles
- Can promote users to admin

## Dashboard Overview
When you first open the application, you'll see the Dashboard with:
- **Statistics Cards**: Quick overview of total students, attendance, and fees
- **Quick Access Cards**: Direct links to main features

## Main Features

### üìù Adding a New Student

1. Click **"Add Student"** from the sidebar or dashboard
2. Fill in the student information:
   - **Student ID** (required) - Unique identifier
   - **Full Name** (required)
   - **Date of Birth** (required)
   - **Grade/Class** (required) - Select from dropdown
   - **Contact Number** (required)
   - **Email** (optional)
   - **Home Address** (optional)
   - **Guardian Name** (optional)
   - **Guardian Contact** (optional)
   - **Enrollment Date** (required) - Defaults to today
3. Click **"Submit"** to save
4. Click **"Reset"** to clear the form

**Tips:**
- Student ID must be unique
- All required fields are marked with a red asterisk (*)
- You'll see a success message when the student is added

### üë• Viewing Students List

1. Click **"Students List"** from the sidebar
2. Use the features:
   - **Search Bar**: Type student name or ID to search
   - **Grade Filter**: Select a specific grade to filter
   - **Sort Columns**: Click column headers to sort
   - **Pagination**: Navigate through pages if you have many students

**Tips:**
- Search works in real-time
- You can combine search and filter
- Click column headers multiple times to toggle sort order

### ‚úÖ Recording Attendance

1. Click **"Attendance"** from the sidebar
2. Select the date using the date picker
3. Filter by grade if needed
4. For each student, select their status:
   - **Present** (green)
   - **Absent** (red)
   - **Late** (yellow)
   - **Excused** (gray)
5. Use **"Bulk Mark"** buttons to mark all visible students at once
6. Click **"Save Attendance"** to save your changes

**Tips:**
- You can mark attendance for any date (past or future)
- Bulk actions only affect currently visible students
- The system prevents duplicate records for the same student on the same date

### üí∞ Managing Fees

1. Click **"Fees"** from the sidebar
2. To add a new fee:
   - Click **"Add Fee Record"**
   - Select student, fee type, amount, and due date
   - Click **"Add Fee Record"** to save
3. To update a payment:
   - Click **"Update"** button next to a fee record
   - Enter the amount paid and payment date
   - Click **"Update Payment"**
4. Use filters to view:
   - All fees
   - Paid fees (green badge)
   - Pending fees (yellow badge)
   - Overdue fees (red badge)

**Tips:**
- Fee types include: Tuition, Library, Sports, Laboratory, Transportation, Other
- Payment status automatically updates based on amount paid
- Use search to quickly find a student's fees

## Color Guide

- **Blue**: Primary actions and active items
- **Teal**: Secondary actions and accents
- **Green**: Success, paid status, present
- **Yellow**: Warnings, pending status, late
- **Red**: Errors, overdue status, absent
- **Gray**: Neutral, excused status

## Tips for Best Experience

1. **Regular Updates**: Mark attendance daily for accurate records
2. **Search First**: Use search before scrolling through long lists
3. **Filters**: Combine search and filters for precise results
4. **Bulk Actions**: Save time with bulk attendance marking
5. **Check Dashboard**: Start each session by reviewing dashboard statistics

## Common Tasks

### Task: Find a specific student
1. Go to Students List
2. Type name or ID in search bar
3. Results appear instantly

### Task: Mark today's attendance
1. Go to Attendance
2. Date is already set to today
3. Mark each student's status
4. Click Save Attendance

### Task: Check pending payments
1. Go to Fees
2. Select "Pending" from status filter
3. View all pending payments

### Task: Update a payment
1. Go to Fees
2. Find the fee record
3. Click Update button
4. Enter payment details
5. Click Update Payment

### Task: Manage Users (Admin Only)
1. Go to Admin Dashboard
2. View all registered users
3. Use dropdown to change user roles
4. Monitor user statistics

## Security Tips

1. **Keep Your Password Safe**: Never share your login credentials
2. **Logout When Done**: Always logout on shared computers
3. **Strong Passwords**: Use passwords longer than 6 characters
4. **Report Issues**: Contact an administrator if you notice any problems

## Need Help?

If you encounter any issues:
- Check that all required fields are filled
- Ensure Student IDs are unique
- Verify dates are in correct format
- Look for error messages (red notifications)
- Success messages appear in green
